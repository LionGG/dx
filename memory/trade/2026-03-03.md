# 2026-03-03 交易心理分析 - 错失航运股

## 用户描述
**时间**: 2026-03-02 盘中
**标的**: 招商南油、中远海能
**机会**: 美伊冲突利好航运股，两票回调时多次想买入
**结果**: 未动手，两票双双涨停
**情绪**: 略微遗憾

## 心理分析

### 1. 错失恐惧(FOMO)的反向表现
通常FOMO是追高风险，但这里是**错失后的遗憾**。这说明：
- 判断是对的（美伊冲突→航运利好）
- 执行力不足（有念头但没动手）

### 2. "规则束缚"心理
用户提到："中天账户不想打野"
- 给自己设了规则：这个账户不做短线/打野
- 但规则成了束缚，挡住了合理机会

### 3. "分心成本"
用户提到："当时正在忙着写量化的程序"
- 注意力被占用，无法及时决策
- 等回过神来，机会已错过

### 4. 账户分工混乱
- 有专门的"打野量化账户"，但盘中没想起来用
- 账户功能划分清晰，但执行时没切换思维

| 问题 | 分析 |

| 执行力 | ❌ 有念头无行动，典型的"想而不做" |

| 情绪影响 | 轻度遗憾，但未影响后续决策（好事） |

## 改进建议

### 1. 区分"纪律"和"僵化"
- 纪律是：不追无逻辑的热点
- 僵化是：明明有逻辑还不动
- **建议**: 给"打野账户"设定明确的启用条件，而非完全不用

### 2. 盘中专注度管理
- 写程序/深度工作时，关闭交易软件或设价格提醒
- 或反向：交易时间专心交易，盘后写程序

### 3. 机会快速评估清单
盘中出现念头时，30秒内回答：
1. 逻辑是否成立？（美伊→航运 ✓）
2. 风险可控吗？（回调买入，非追高 ✓）
3. 哪个账户适合？（打野账户 ✓）
4. 仓位多少？（打野仓位，轻仓 ✓）

四个✓就动手，有✗就放弃，不纠结。

### 4. 错失后的情绪处理
- 这次"略微遗憾"是健康的，说明没陷入过度自责
- 警惕：不要让"怕再错过"变成下次盲目追高的理由

## 一句话总结

**判断正确但执行断裂——规则成了束缚，分心是导火索。建议建立"快速评估清单"，30秒内决策，不纠结。**
