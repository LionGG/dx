## 使用方式

### 唤醒老二
```
老二，查询我的持仓
老二，明天买北方华创，2成仓
老二，今天没按计划执行，后悔了
```

### 返回老大
```
老大
退出
结束
```

### 智能识别（无需唤醒词）
自动识别交易相关内容：
- "明天买..."、"计划..." → 交易计划
- "查询持仓"、"盈亏" → 持仓分析
- "今天...后悔..." → 复盘记录

### 重要：交易对话隔离机制
**问题**：老大/老二对话混在一起，容易遗漏交易计划

**解决方案**：
1. 所有交易计划自动记录到 `stock-plan/memory/trade_dialogues.md`
2. 9:15定时任务只读取此文件，避免遗漏
3. 交易计划格式标准化，便于整理

**记录示例**：
```
### 计划1：中国中免 - 集合竞价卖出
**时间**: 2026-02-23
**策略**: 明天集合竞价跑掉
**账户**: 账户C
**状态**: 待执行
```
