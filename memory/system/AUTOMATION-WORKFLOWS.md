# AUTOMATION-WORKFLOWS.md — 工作流自动化配置

## Status
- **Enabled:** ✅
- **Primary Use:** 股市数据全流程自动化
- **Error Handling:** 自动重试 + 通知

## 核心工作流

### 1. 股市数据日终流程
```
16:05 个股数据抓取
    ↓ (成功)
16:10 MA50占比计算
    ↓ (成功)
16:20 短线情绪数据抓取
    ↓ (成功)
16:25 短线情绪研判 (AI分析)
    ↓ (成功)
16:30 部署通知 (GitHub+截图)
    ↓ (完成)
飞书群通知
```

### 2. 错误处理流程
```
任务失败
    ↓
自动重试 (最多3次)
    ↓
仍失败 → 发送飞书告警
    ↓
记录到 ERROR_PREVENTION.md
```

### 3. 交易相关流程
```
09:10 交易计划分析
    ↓
推送交易计划 + 心理分析
    ↓
15:00 收盘后
    ↓
接收持仓截图
    ↓
对比计划 vs 执行
    ↓
生成执行分析报告
```

## 自动化规则

| 条件 | 动作 |
|------|------|
| 任务状态 = error | 重试3次 → 告警 |
| 情绪指数 < 30 | 自动提示"冰点机会" |
| MA50占比变化 > 5% | 自动提示"趋势变化" |
| 定时任务未执行 | 自动检查并补执行 |

## 集成工具

- **OpenClaw Cron:** 定时触发
- **飞书:** 消息推送
- **墨问:** 报告发布
- **SQLite:** 数据存储
- **GitHub Pages:** 网页部署
