# A-Share Sector 板块分析项目

**项目位置**: `stock/a-share-sector/`

## 核心组件

| 文件 | 功能 |
|------|------|
| `scripts/data_provider.py` | 数据读取模块（从 a-share-warehouse 读取） |
| `scripts/generate_sector_report_final.py` | 分析引擎（SectorAnalyzer 类） |
| `scripts/publish_to_mowen.py` | 墨问发布模块 |
| `reports/` | 生成的报告目录 |

## 分析框架

### 主线板块判定
- **资金深度**: 涨停股总成交额
- **涨停梯队**: 最高连板数
- **封单强度**: 平均封单金额
- **中军力量**: 成交额>10亿且涨幅>3%的个股
- **分数门槛**: ≥50分，且处于主升/确认/发酵期

### 发酵板块判定
- 有晋升主线可能
- **一日游识别**: 资金深度<30亿 或 无高度龙头

### 风险板块判定
- 退潮期/分歧期
- 跌幅>3%
- 前期炒作充分

## 数据维度

| 维度 | 指标 |
|------|------|
| 板块指数 | 涨幅、成交额、换手率 |
| 涨停股 | 家数、连板高度、封单金额、涨停时间 |
| 中军股 | 成交额>10亿且涨幅>3% |
| 龙虎榜 | 机构/游资参与度 |

## 关键词匹配策略

- **同义词扩展**: 航海装备→船舶/造船/海洋装备
- **上下游关联**: 新能源→光伏/锂电/储能
- **模糊匹配**: 其他数字媒体→数字媒体

## 报告结构

```
一、主线板块（重仓参与）
  **判定标准**: 资金深度+涨停梯队+封单强度达标，分数≥50
  - 板块表现
  - 涨停分析
  - 龙头梯队
  - 中军力量
  - 主线判定

二、发酵板块（小仓试错）
  **判定标准**: 有晋升主线可能，需判断持续性
  - 一日游判断
  - 操作建议

三、风险板块（坚决回避）
  **判定标准**: 退潮期/分歧期，或跌幅>3%，或前期炒作充分
  - 风险特征
  - 板块表现
```
