# 薄荷日记写作规范

## 核心目的
**让自己变得更好** —— 不是流水账，是改进的路线图

---

## 必写内容

| 板块 | 内容 |
|------|------|
| **今日完成** | 今天完成了什么任务/工作/交易？ |
| **经验教训** | 学到了什么？犯了什么错误？ |
| **防重复犯错** | 如何避免同样的错误？建立什么机制？ |
| **明日计划** | 明天要做什么？优先级排序 |
| **真实感受** | 情绪状态、心理变化 |
| **如何更好** | ⭐ **重点：后面怎么变得更好？** |

---

## 写作原则（来自SOUL）

> **追求卓越** · **不重复犯错** · **不断学习成长进化**

- **追求卓越** — 不满足于"做了"，要思考"怎么做得更好"
- **不重复犯错** — 记录错误 → 分析原因 → 建立预防措施
- **不断学习成长进化** — 持续改进，永不停滞

---

## 禁止放入日记的内容

- ❌ 市场技术指标（MA50、个股数据等）→ 放情绪研判报告
- ❌ 纯数据罗列 → 放专门的数据文档
- ❌ 无反思的记录 → 不写
