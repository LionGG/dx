# 美股收盘推送核查清单

## 执行前核查
- [ ] 确认今天是周六（美股周五收盘数据）
- [ ] 搜索关键词包含 "美东时间3月X日" 或 "周五收盘"
- [ ] 确认文章中的日期是美东时间，不是北京时间/文章发布时间

## 数据核查
- [ ] 三大指数涨跌幅是否合理？（与昨日相比差异>2%需二次验证）
- [ ] 中概股金龙指数涨跌方向与大盘是否一致？（通常有关联）
- [ ] 核对道指点位：正常范围 40,000-50,000（超出范围需验证）

## 常见错误预防
1. **日期混淆**：文章发布日期 ≠ 数据日期
   - 第一财经：北京时间3月6日发布的可能是美东3月5日数据
   - 正确做法：找明确写"美东时间3月6日周五收盘"的报道

2. **时差计算**：
   - 美东时间周五收盘 = 北京时间周六早上
   - 美东时间周四收盘 = 北京时间周五早上

3. **数据源验证**：
   - 优先选择：新浪财经、第一财经、华尔街见闻等主流财经媒体
   - 交叉验证：至少两个来源数据一致

## 错误示例
- ❌ 2026-03-07错误：把美东3月5日数据（道指47,954）当成3月6日数据
- ✅ 正确数据：美东3月6日道指47,501
