# 交易助手唤醒规则（最终版）

## 方式1：智能识别（默认）
自动判断消息类型，无需唤醒词

**强信号词**：
- 操作：买、卖、进、出、加仓、减仓、止损、止盈
- 查询：持仓、盈亏、账户、仓位
- 复盘：今天、后悔、没按、应该

**示例**：
- "明天中免低开就加仓" → 交易助手
- "今天没买后悔了" → 交易助手
- "OpenClaw怎么配" → 主助手

---

## 方式2：唤醒词（精准呼叫）

### 唤醒词
- **"老二"**（交易助手昵称）
- "交易助手"

### 使用
```
老二，明天北方华创突破28就进
```
或
```
老二
↓
在，请说
↓
明天北方华创突破28就进
```

### 退出
说 "退出"、"结束" 或 "老大" 返回主助手

---

## 角色设定

| 角色 | 名字 | 职责 |
|------|------|------|
| 主助手 | 老大 | 日常对话、综合任务 |
| 交易助手 | 老二 | 交易记录、持仓分析、复盘 |

---

## 示例对话

**场景1：智能识别**
```
你：明天计划买北方华创
→ 老大识别 → 转老二处理
```

**场景2：唤醒词**
```
你：老二
老二：在，请说
你：查询持仓
老二：📊 三账户...
你：老大
→ 返回主助手
```

**场景3：连续指令**
```
你：老二，明天买北方华创，2成仓，27止损
老二：✓ 计划已记录...
```
