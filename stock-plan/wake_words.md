# 交易助手唤醒词

## 唤醒词
- "交易助手"
- "小交易"
- "助手"

## 使用方式
1. 先说唤醒词
2. 接着说内容

## 示例

**方式1：连续说**
```
交易助手，明天北方华创突破28就进
```

**方式2：先唤醒**
```
交易助手
↓
（助手回复：在，请说）
↓
明天北方华创突破28就进
```

**方式3：带前缀**
```
[交易] 明天北方华创突破28就进
```

## 识别逻辑
if "交易助手" in message or "小交易" in message:
    → 唤醒交易助手
    → 后续内容转给 trade-assistant

## 退出
说 "退出" 或 "结束" 返回主助手
