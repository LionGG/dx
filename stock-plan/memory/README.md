# 持仓记录存储规范

## 目录结构

```
stock-plan/memory/
├── positions/              # 每日收盘持仓记录（核心）
│   ├── 2026-02-25.md      # 账户A/B/C持仓明细
│   ├── 2026-02-26.md      # 账户A/B/C持仓明细
│   └── 2026-02-27.md      # 每日更新
├── plans/                  # 交易计划记录
│   └── 2026-02-27.md      # 当日交易计划
├── reviews/                # 复盘记录
│   └── 2026-02-26.md      # 每日复盘
└── README.md              # 本文件
```

## 文件命名规则

| 类型 | 命名格式 | 示例 |
|------|---------|------|
| 收盘持仓 | `YYYY-MM-DD.md` | `2026-02-26.md` |
| 交易计划 | `YYYY-MM-DD_plan.md` | `2026-02-27_plan.md` |
| 复盘记录 | `YYYY-MM-DD_review.md` | `2026-02-26_review.md` |

## 持仓记录格式

```markdown
# YYYY-MM-DD 收盘持仓

## 账户A：杠杆埋伏号
| 项目 | 数值 |
|------|------|
| 总资产 | X万 |
| 当日盈亏 | X万 (X%) |

### 持仓明细
| 代码 | 名称 | 盈亏比 | 仓位占比 |

## 账户B：均衡配置号
...

## 账户C：核心资产号
...

## 三账户汇总分析
...
```

## 查找最新持仓的方法

按文件修改时间排序取最新：
```bash
ls -lt stock-plan/memory/positions/*.md | head -1
```

## 历史记录归档

- 保留最近30天的持仓记录在 `positions/`
- 超过30天的移至 `positions/archive/`
