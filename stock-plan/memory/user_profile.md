## 关键发现

### 1. 盈利密码
**账户C成功关键**：
- 中国中免大赚90万（+31%）
- 无高杠杆压力
- 持仓更分散

### 2. 亏损陷阱
**账户A失败原因**：
- 高杠杆放大亏损
- 券商过度集中
- 全仓浮亏无对冲

### 3. 风格建议
| 账户 | 建议 |
|------|------|
| 账户A | **减仓降杠杆**，学习账户C的分散策略 |
| 账户B | 保持，优化选股 |
| 账户C | **主力账户**，可加大投入 |
